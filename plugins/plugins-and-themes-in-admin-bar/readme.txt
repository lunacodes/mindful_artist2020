Adds Theme and Plugin menus in the admin bar.



== Description ==



This plugin is for WordPress developers who need to install, upload and edit themes or plugins often. This makes it easy to access these by adding the menus in the admin bar.



== Installation ==

1. Upload the plugins and themes in admin bar folder to /wp-content/plugins/.

2. Activate the plugin through the "Plugins" menu in WordPress.
